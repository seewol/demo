# My Demo Project

Spring Boot + Gradle 학습용 프로젝트! 🎉  
이커머스 관련 기본 API를 구현하며 스프링 부트와 JPA를 연습합니다.


---


## 🚀 Tech Stack
- Java 17
- Spring Boot 3.4.8
- Spring Data JPA
- Gradle


---


## 📂 Project Structure
```
src
├─ main
│ ├─ java/com/jeeeun/demo
│ │ ├─ controller (API 엔드포인트)
│ │ ├─ service (비즈니스 로직)
│ │ ├─ domain (엔티티)
│ │ └─ repository (JPA Repository)
│ └─ resources
│   └─ application.properties (환경 설정)
└─ test (테스트 코드)
```
