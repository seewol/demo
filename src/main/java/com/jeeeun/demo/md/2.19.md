##   구글 로그인 연동 가이드 : 숙제

1. 사전 준비 — Google Cloud Console 설정

1. https://console.cloud.google.com/ 접속
2. 새 프로젝트 생성 (또는 기존 프로젝트 선택)
3. APIs & Services > OAuth consent screen 설정
   - User Type: External 선택
   - 앱 이름, 이메일 등 기본 정보 입력
   - Scopes에 email, profile, openid 추가
4. APIs & Services > Credentials > Create Credentials > OAuth 2.0 Client IDs
   - Application type: Web application (또는 Android/iOS)
   - Authorized redirect URIs: 클라이언트 앱의 콜백 URL 등록
   - 생성된 Client ID와 Client Secret 메모


### 플랫폼별 SDK:
- Android: https://developer.android.com/identity/sign-in/credential-manager-siwg
  또는 Google Sign-In
- iOS: https://developers.google.com/identity/sign-in/ios
- Web: https://developers.google.com/identity/gsi/web

### 구글 api 호출
infra/http-client/src/.../google/GoogleOAuth2Client.kt
``` shell
  @FeignClient(name = "googleApiClient", url = "\${service.google.baseUrl}")
  interface GoogleOAuth2Client {
      @GetMapping("/oauth2/v2/userinfo")
      fun getUserInfo(@RequestHeader("Authorization") authorization: String):
  GoogleOAuth2UserInfo
  }

```

- https://www.googleapis.com/oauth2/v2/userinfo에 Bearer <accessToken> 헤더로
  요청
- 응답: { id, email, name, picture, verified_email, ... }
