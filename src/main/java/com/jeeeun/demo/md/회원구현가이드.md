## 회원가입 구현 학습 가이드
회원가입 흐름을 아래에서 위로(DB → Domain → Application → API) 순서로 단계별 구현한다.

---

## 사전 지식: Google OAuth2 흐름

이 프로젝트는 **Google OAuth2 소셜 로그인** 방식을 사용한다.

### Google OAuth2 관련 API
https://developers.google.com/identity/protocols/oauth2?hl=ko
인증 페이지(https://accounts.google.com/o/oauth2/v2/auth) → Authorization Code
수신 → Access Token 교환(https://oauth2.googleapis.com/token) → 사용자 정보
조회(https://www.googleapis.com/oauth2/v2/userinfo) 전체 흐름이 설명
